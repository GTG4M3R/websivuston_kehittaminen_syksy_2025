Module 5 - Responsive design

- a responsive layout with two columns on larger screens and a single column on smaller screens
- Added an image that resizes automatically to fit its container
- Used media queries to change the background color of a div based on the screen width
- Created a navigation bar that collapses into a menu button on smaller screens

    - Browsers developer tools
    - Simulated different screen sizes
